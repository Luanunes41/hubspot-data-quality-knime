# hubspot-data-quality-knime

Projeto KNIME para tratamento de dados de contatos antes da importação no HubSpot.

Funcionalidades:
- Remoção de espaços
- Normalização de emails
- Validação de emails
- Remoção de duplicados
- Padronização de telefones
- Score de qualidade dos contatos
- Exportação para Excel
